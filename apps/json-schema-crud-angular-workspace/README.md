# JSON Schema CRUD Angular Workspace

Angular workspace that combines host shell and microfrontend-oriented examples around a JSON schema CRUD and mapper experience.

## Stack

- Angular workspace
- TypeScript
- Playwright
- Microfrontend patterns

## Main Purpose

- Demonstrate a multi-app Angular workspace
- Show host shell and remote/module composition patterns
- Provide a reusable front-end architecture for schema-focused platforms

## Run

1. Install dependencies with `npm install`
2. Start the desired app or shell using the workspace scripts

## Folder Role In This Monorepo

Use this folder as a boilerplate when starting a larger Angular workspace with multiple apps, remotes, and shared libraries.
