# JSON Mapper Angular App

Angular application focused on visually mapping source JSON/schema structures into target schemas.

## Stack

- Angular
- TypeScript
- Playwright

## Main Purpose

- Import source and target schemas
- Map fields visually
- Persist mapping metadata to a backend API

## Run

1. Install dependencies with `npm install`
2. Start the app with `npm start` or `npm run dev`

## Folder Role In This Monorepo

Use this folder as a boilerplate when starting a standalone Angular UI for JSON mapping or schema transformation workflows.
