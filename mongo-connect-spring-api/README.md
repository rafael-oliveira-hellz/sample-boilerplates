# Mongo Connect Spring API

Spring Boot API example showing MongoDB connectivity plus CRUD-style modules for business entities such as event types, partners, and products.

## Stack

- Java
- Spring Boot
- MongoDB
- Docker Compose

## Main Purpose

- Demonstrate MongoDB connection setup
- Expose REST endpoints with validation and pagination
- Provide reusable structure for entity-based Spring APIs

## Run

1. Review environment variables and configuration files
2. Start dependencies if needed with Docker Compose
3. Run the project with Maven

## Folder Role In This Monorepo

Use this folder as a boilerplate when starting a Spring Boot service that needs MongoDB, REST CRUD endpoints, and production-oriented configuration patterns.
