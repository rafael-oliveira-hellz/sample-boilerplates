# JSON Schema CRUD Spring API

Spring Boot API focused on saving and loading JSON schema configuration documents.

## Stack

- Java
- Spring Boot
- Maven

## Main Purpose

- Persist source and target schema configurations
- Expose endpoints for saving and loading mapper configuration payloads
- Serve as a compact backend reference for schema-driven workflows

## Run

1. Build with Maven
2. Start the API locally
3. Connect a frontend client to the exposed endpoints

## Folder Role In This Monorepo

Use this folder as a boilerplate when starting a small Spring Boot CRUD API for structured JSON configuration storage.
